如何在 Windows 上安装 OBS YOLO 检测器插件
===============================================

方法一：自动安装（推荐）
-------------------------
1. 解压下载的 zip 压缩包
2. 双击 "Install Into Here.lnk" 快捷方式
3. 如果提示找不到位置，手动将整个文件夹复制到 OBS 插件目录

方法二：手动安装
-----------------
1. 解压下载的 zip 压缩包
2. 将 `obs-backgroundremoval` 文件夹复制到 OBS Studio 的插件目录
3. 默认插件目录是：`C:\ProgramData\obs-studio\plugins\`
4. 如果 `C:\ProgramData\obs-studio` 或 `C:\ProgramData\obs-studio\plugins` 文件夹不存在，请先手动创建

验证安装
---------
1. 启动 OBS Studio
2. 在源上右键 -> 滤镜
3. 点击 "+" 号，查看是否有 "YOLO检测器" (YOLODetector) 选项
4. 如果有，说明安装成功

使用说明
---------
1. 添加 YOLO 检测器滤镜到视频源
2. 在设置中选择 YOLO 模型文件 (.onnx)
3. 点击"开始推理"按钮开始检测
4. 可以调整置信度阈值、NMS 阈值等参数
5. 勾选"显示边界框"可以在画面上看到检测结果
